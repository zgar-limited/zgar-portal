# Zgar Portal 项目注意事项

## 币种设置

**重要：整个项目使用美元（USD）作为币种，不是人民币！**

### 显示规则

- **余额显示**：使用 `$` 符号，不是 `¥`
- **格式**：`${amount.toFixed(2)}`
- **示例**：`$1250.50` ✓ / `¥1250.50` ✗

### 需要修改的地方

#### 1. components/dashboard/Sidebar.tsx
```typescript
// 第81行 - 余额显示
${customer?.zgar_customer?.balance?.toFixed(2) || "0.00"}
```

#### 2. components/dashboard/MyAccount.tsx
```typescript
// 第150行 - 余额卡片
${stats.balance.toFixed(2)}
```

#### 3. 其他所有涉及金额显示的地方
- 搜索代码中的 `¥` 符号，全部替换为 `$`
- 订单详情页
- 购物车页面
- 产品列表页
- 结账页面

### 注意事项

⚠️ **不要使用人民币符号 `¥`**
⚠️ **始终使用美元符号 `$`**
⚠️ **金额格式：两位小数，如 `$99.99`**
